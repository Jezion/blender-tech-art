# blender-tech-art

Python scripts for Blender — asset pipeline automation.  
Work in progress | Technical Artist career transition.

## Scripts

| Script | Description |
|--------|-------------|
| `create_demo_scene.py` | [create a scene with 5 objects with random materials] |
| `batch_material_replace.py` | [apply basic material on selected objects] |
| `material_utils.py` | [material operators shared between scripts]

## Setup

1. Blender 5.x
2. Open Script Editor in Blender
3. Load script and run with Alt+P
